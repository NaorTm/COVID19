# COVID-19 Patient Management API (Node.js)

Express API that manages COVID-19 patients, encounters, lab tests, and statistics using an in-memory data model.

## Features
- CRUD endpoints for patients and lab tests
- Encounter and route tracking
- City-wide statistics and isolation data
- Validation with express-validator

## Run
```sh
npm install
npm start
```

Server runs on http://localhost:3000

## Tests
```sh
npm test
```
